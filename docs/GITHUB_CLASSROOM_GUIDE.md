# GitHub Classroom Guide

This guide provides instructions on how to use GitHub Classroom for your assignments.

## 1. Getting Started
- Accept the assignment via the provided link.
- Clone the repository to your local machine.

## 2. Working on Assignments
- Create a new branch for your features or fixes.
- Commit your changes regularly with descriptive messages.
- Push your changes to the remote repository.

## 3. Submitting
- Ensure all required files are present and pass the CI checks.
- Submit your work by pushing to the default branch (usually `main`).
